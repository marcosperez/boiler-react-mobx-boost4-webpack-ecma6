## TEMPLATE 

Template inicial para generar aplicaciones React-Mobx

### Contenido:
- boostrap 4
- lodash 4.17.4
- jquery
- tether
- fontawasome 4.7
- react
- mobx
- eslint
- babel

### Comandos

- *npm start:* Lanza aplicacion en navegador con hot-reload
- *npm build:* Genera build de la aplicacion.
